# Getting Help & Support

Thank you for using VibeCoderz! If you need assistance, have questions, or want to connect with other builders, please refer to the resources below.

---

## 💬 Discussion Channels

- **GitHub Discussions**: The primary home for general Q&A, idea brainstorming, and showcase sharing is our [GitHub Discussions page](https://github.com/mohdmohsinrizvi/vibecoderz/discussions).
- **Twitter/X**: You can follow and tag the founder at **[@mohdmohsinrizvi](https://twitter.com/mohdmohsinrizvi)** for quick updates and platform announcements.

---

## 🐛 Bug Reports & Feature Requests

- **Reporting Bugs**: If you encounter unexpected behavior, please search the active issues first. If it's a new bug, file a report using the [Bug Report template](https://github.com/mohdmohsinrizvi/vibecoderz/issues/new?template=bug_report.md).
- **Requesting Features**: We love learning about how VibeCoderz can be improved! Open a request using the [Feature Request template](https://github.com/mohdmohsinrizvi/vibecoderz/issues/new?template=feature_request.md).

---

## ✉️ Direct Contact

For administrative inquiries, sponsorship offers, or Code of Conduct reports, you can reach the maintainer privately at **[mohsinrizvi.dev@gmail.com](mailto:mohsinrizvi.dev@gmail.com)**.
