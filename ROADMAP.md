# VibeCoderz Product Roadmap

This roadmap outlines the past, present, and future trajectory of VibeCoderz. Contributions that align with these milestones are highly encouraged.

---

## 🏁 Phase 1: MVP Stability (Completed)
- [x] **Database Schema Setup**: Supabase database with projects, users, reactions, and comments tables.
- [x] **Project Submission**: Fully functioning form allowing builders to link github URLs, live demos, and category tags.
- [x] **Reactions Module**: Stateful inline bars allowing dynamic upvote/ship/fire/mindblown counts.
- [x] **Leaderboard**: Calculation of global reputation scores based on submission count and community reaction weights.
- [x] **Premium UI**: Full implementation of Alabaster/Indigo design systems with smooth CSS transitions.

---

## 🤝 Phase 2: Community & Engagement (In Progress)
- [ ] **Threaded Discussions**: Nested comment replies with markdown formatting for code snippets.
- [ ] **User Follows**: A personalized activity feed displaying updates from followed builders.
- [ ] **Weekly Showdowns**: Automatic highlighting of the top 3 projects shipped each week with exclusive homepage ribbons.
- [ ] **Dynamic Empty States**: Illustrative and action-focused empty state designs for feed filters and search queries.

---

## 📈 Phase 3: Growth & Integrations
- [ ] **OAuth Registrations**: Integration with GitHub Auth and Twitter/X login credentials.
- [ ] **GitHub Action Integration**: A reusable VibeCoderz action that automatically posts to the feed when a repository is marked release-ready.
- [ ] **Developer Portfolios**: Auto-generated custom portfolio pages for each builder based on their shipped projects history.
- [ ] **API Access**: Public API tokens allowing developer tools (like Raycast) to search projects and pull stats.

---

## 🔮 Phase 4: Long-Term Vision
- [ ] **Builder Cohorts**: Temporary building sprints / hackathons managed directly inside the platform.
- [ ] **Verified Badges**: Integration of verified status checks based on GitHub accounts or live URL verification.
- [ ] **Job Board Feed**: A premium feed for companies seeking builders with high platform reputation scores.
