## [1.0.0] - 2026-06-13

This is the initial release version prepared for open source collaboration.

### Added
- Complete brand visual overhaul introducing theme-specific **Deep Indigo** and **Subtle Violet** color systems.
- Dynamic theme-aware brand logos (`logo-dark.png` and `logo-light.png`).
- Organic abstract glowing mesh background overlay for premium visual texture.
- Responsive bento grid layout profiles for About / Landing pages.
- Advanced global ranking leaderboards displaying active builder states.
- Dedicated OpenGraph metadata configurations and 1200x630 share card previews.
- Full set of standalone browser favicons, apple touch shortcuts, and PWA manifests.
- CI pipeline workflow targeting automated tests, building, and linting.
- Contributor templates (PR / Bug Report / Feature templates).
- Open source standards documents (`CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, `SECURITY.md`, `SUPPORT.md`, `ROADMAP.md`).
